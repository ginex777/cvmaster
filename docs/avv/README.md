# AVV-Nachweise

Lege hier vor Public-Launch die finalen AVV- oder DPA-Nachweise fuer alle tatsaechlich genutzten Auftragsverarbeiter ab.

Nicht committen:

- Zugangsdaten
- API-Keys
- private Vertragsnummern, falls sie nicht fuer das Team bestimmt sind

Erwartete Nachweise:

- AI-Provider
- Resend oder alternativer Mailprovider
- Paddle
- Hosting/IONOS oder alternativer Infrastrukturprovider
- Crisp oder alternativer Supportanbieter
- Monitoring/Analytics, falls aktiviert
